## \#1. The Physical Layer
Raw bitstream is physically transmitted over a physical medium.


## \#2. The Data Link Layer
Breaks data into frames for transmission at the physical layer. Also manages connection between nodes, including bit errors.

## \#3. The Network Layer
Responsible for routing traffic to its destination based on IP addresses.

## \#4. The Transport Layer
Manages the transmission of data between nodes. The Transmission Control Protocol (TCP) operates at Layer 4.

## \#5. The Session Layer
Manages sessions between nodes.

## \#6. The Presentation Layer
Translate data from network data to the formats expected by an application. Includes encoding and encryption.

## \#7. The Application Layer
Includes protocols designed to transmit data to end-user.Ex: HTTP
